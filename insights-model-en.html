About Insights Discovery

Insights Discovery is a psychometric profiling model developed in Scotland, based on Carl Jung's theory of psychological types. The model uses four color energies to describe behavioral preferences and communication styles.

Each person has access to all four energies, but tends to prefer and use one or two predominantly.
