# LeaderKit

Interactive personality and behavior model simulators (DISC, MBTI, Belbin, etc.).

## Overview

A collection of interactive web-based simulators for popular personality and leadership models. Available in both English and Romanian. Explore how different frameworks categorize and analyze personality traits and team roles.

## Supported Models

- **DISC** - Dominance, Influence, Steadiness, Conscientiousness
- **MBTI** - Myers-Briggs Type Indicator
- **Belbin** - Team Role model
- **OCEAN** - Big Five personality traits
- **PAEI** - Adizes management styles
- **Hogan** - Hogan Personality Inventory
- **Insights** - Insights Discovery

## Usage

Open `index.html` (Romanian) or `index-en.html` (English) in a browser, then navigate to any model simulator.
